⚠ Firebase無料版について

無料版Firebase（Sparkプラン）は、
小〜中規模配信向けです。
目安として 50〜100人程度 の同時利用は、
計算上問題なく動作する想定です。
大規模配信・バズ時は、
Firebase有料版（Blaze）の検討をおすすめします。
🟢 個人配信 → 無料版OK
🟡 中規模配信 → 無料〜Blaze推奨
🔴 バズ級配信 → Blaze推奨

🔥 Firebase設定ガイド

初めてでも分かる版

① Firebase Console を開く

設定画面内の「Firebase Console」ボタンを押す。
ここで Googleログイン するだけでOK

② 「プロジェクトを作成」を押す

名前は何でもOK。
例：StreamStaff / HokutoSS など。
「プロジェクトを作成」ボタンを押す画面

③ Webアプリを追加する

「</>」みたいなWebマークを押す。
iOS / Android / Web の中から Web を選択

④ firebaseConfig をコピー

すると Firebase が英語のコードを表示する。
その中の apiKey や projectId を StreamStaff にコピペ。
const firebaseConfig = { ... }

⑤ StreamStaffへ貼る

各欄に貼り付けて「Firebase設定保存」を押す。
接続テストが通ればOK！
